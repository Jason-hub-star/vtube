# 페이셜 테스트 체크리스트 (H2 검수 시트)

Updated: 2026-06-11 · 업계 기준(캘리브레이션 가이드 + Live2D 표준 파라미터) 기반 · 매 캐릭터 재사용

리깅 앱(슬라이더)과 웹캠 드라이브 양쪽에서 확인한다. **항상 중립 보정부터.**

## 1. 머리 (슬라이더 → 웹캠)

| 항목 | 확인 방법 | 합격 기준 |
|---|---|---|
| 좌우 회전 (AngleX ±30) | 고개 좌우 | 얼굴·앞머리 이동 + 목 부분 추종, **턱·목 이음새 없음** |
| 뒷머리 동행 (CHAIN-001) | 고개 좌우 완전 회두 | **뒷머리·어깨 가닥이 머리와 한 덩어리로 이동** — 가슴에 정지 잔류 조각 없음 |
| 상하 까딱 (AngleY ±30) | 끄덕임 | 얼굴 상하 + 머리카락 세로 출렁 |
| 기울임 (AngleZ ±30) | 갸웃 | 머리 회전, 목 비틀림 없음 |
| 목 분리 | X+Y 동시 최대 | 목 밴드 투명 픽셀 0 (스윕 수치 검사 자동화됨) |

## 2. 눈

| 항목 | 확인 방법 | 합격 기준 |
|---|---|---|
| 깜빡임 (EyeL/ROpen) | 눈 감았다 뜨기 | **연속 정점 키폼** 아치 감김(단계 전환 없음), 0.27 자연 감김, 좌우 독립(윙크) |
| 상하 감김 (EYE-NATURAL-001) | 천천히 감기 | 윗꺼풀 하강 + **아랫꺼풀·아랫속눈썹 상승**(컬럼 눈높이 20%), 닫힘선은 중심 아래 아치 — 위에서만 내려오는 셔터식이면 FAIL |
| 잔상 없음 (EYE-NATURAL-002) | 슬라이더로 천천히 감기 | 감는 중 **이전 단계 눈이 흐릿하게 겹쳐 보이면 FAIL** (크로스페이드 잔상 — `validate_blink_keyforms.py` 자동화됨) |
| 시선 (EyeBallX/Y) | 눈동자 굴리기 | 홍채가 흰자 안에서만 이동 (클리핑) |

## 3. 입 — "살아있음의 핵심" (자연스럽게 말하며 테스트, 과장 금지)

| 항목 | 확인 방법 | 합격 기준 |
|---|---|---|
| 개방 단계 (MouthOpenY 0→0.85) | 평소처럼 말하기 | 다묾=원본, 단계 자연, **턱선 항상 보임** (가드 자동화됨) |
| 입 모양 일관성 | "아-이-우" | 상태 전환이 튀지 않음 (겹침 크로스페이드) |

## 4. 눈썹·표정

| 항목 | 확인 방법 | 합격 기준 |
|---|---|---|
| 눈썹 상하 (BrowL/RY) | 슬라이더 (트래킹 후속) | 자연스러운 ±8px, 이마에 잔상 없음 |
| 표정 의도성 | 웹캠 일반 대화 | 의도 않은 표정 발동 없음 (표정 스팸 금지) |

## 5. 몸·물리

| 항목 | 확인 방법 | 합격 기준 |
|---|---|---|
| 상체 추종 (BodyAngleX/Y) | 고개 돌림 | 상체가 은은히 따라 기움 (±10) |
| 머리-몸 탑승 (CHAIN-001) | 고개 돌림 | 몸 스웨이 시 머리가 함께 운반 — 머리-몸 상대 슬립 없음 |
| 목-어깨 접합 (등변위 원리) | 회두+스웨이 복합 | 접합부에 단차·이중 윤곽 없음 (목·가슴·배경판 등변위) |
| 뒷머리 수동 (HairBack) | 슬라이더 ±1 | 뒷머리만 좌우 스윙 (얼굴 부동) |
| 호흡 (Breath) | 가만히 | 가슴 미세 오르내림 |
| 머리 물리 | 고개 빠르게 흔들기 | 앞머리 가볍게·뒷머리 묵직하게 출렁 후 감쇠 정지 |

## 6. 성능

| 항목 | 확인 방법 | 합격 기준 |
|---|---|---|
| 프레임 | 드라이브 FPS 표시 | 30fps 이상, 끊김 체감 없음 (pixi 기본 — 렌더 상한 60fps, 미달 시 트래킹 쪽 의심) |
| 적용 파라미터 | 드라이브 "적용" 표시 | 9 이상 |

## 자동화된 검사 (수동 확인 전 선행)

- validator PASS + 스윕 스모크 (전 상태 해시 상이) + 목 투명픽셀 검사 + T3 재생 스모크
- 렌더 백엔드: `run_mesh_deform_verify.py --renderer pixi` 5/5 (backend_match 포함) + `run_rig_perf_test.py --renderer pixi`
- 보류 항목: 옆모습(±30° 초과), 볼 부풀리기·혀 추적(VBridger급), MouthForm 표정 연동
