# imagegen VTuber 파츠 생성 한계 테스트 001

작성일: 2026-06-02

## 목적

Codex imagegen을 2D VTuber 제작 파이프라인에 사용할 때 어디까지 맡길 수 있는지 확인한다.

검증 대상:

```text
1. canonical 정면 원화 생성
2. 같은 캐릭터의 파츠 시트 생성
3. underpaint 생성
4. Live2D/PSD 레이어 후보로 쓸 수 있는지 평가
```

## 판정 기준

### A. 일관성

```text
머리색, 눈색, 의상, 액세서리, 선 굵기, 얼굴 비율이 유지되는가
```

### B. 파츠 적합성

```text
눈, 입, 머리카락, 몸/의상이 개별 파츠로 분리되어 보이는가
파츠가 서로 겹쳐 있지 않은가
각 파츠가 alpha cutout 후보로 쓸 수 있는가
```

### C. Underpaint 적합성

```text
앞머리 뒤 이마
눈동자 이동용 눈 흰자
입 벌림용 입안/치아/혀
목/턱 뒤 빈 영역
```

### D. 후처리 필요도

```text
그대로 사용 가능
마스크/정렬 후 사용 가능
참고용으로만 사용 가능
폐기
```

## 실험 프롬프트

### Test A: canonical_front

정면 half-body anime VTuber 원화. 흰색 또는 단색 배경. 파츠 분리하기 쉬운 선명한 실루엣. 앞머리, 눈, 입, 상의가 명확히 보이는 디자인.

### Test B: parts_sheet

Test A와 같은 캐릭터의 Live2D 제작용 파츠 시트. 얼굴 베이스, 눈 흰자, 홍채, 동공, 하이라이트, 속눈썹, 눈썹, 닫힌 입, 열린 입 내부, 치아, 혀, 앞머리 조각, 옆머리, 뒷머리, 목, 몸통, 상의 파츠를 흰 배경 위에 따로 배치.

### Test C: underpaint_sheet

Test A와 같은 캐릭터의 가려진 영역 보충 시트. 앞머리 없는 얼굴 베이스, 이마, 머리 뒤쪽, 목/턱 아래, 입안 내부, 눈 흰자 확장 영역을 정리.

## 기대 결과

imagegen이 모든 파츠를 완벽히 분리해주지는 못할 가능성이 높다. 성공 기준은 "완성 파츠"가 아니라 "Codex가 후처리할 수 있는 충분히 구조화된 후보 이미지"를 얻는 것이다.
